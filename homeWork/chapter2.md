# 第二章作业答案

## 1. 多层感知机和感知机的区别是什么，为什么会有这样的区别？

### 主要区别：

**（1）结构差异**
- **感知机**：只有输入层和输出层两层结构，是单层神经网络
- **多层感知机（MLP）**：包含输入层、至少一个隐层和输出层，是深层神经网络

**（2）解决非线性问题的能力**
- **感知机**：只能解决线性可分问题（如AND、OR问题），无法解决异或（XOR）等非线性问题
- **多层感知机**：通过隐层的非线性变换，可以解决任意复杂的非线性问题

**（3）激活函数**
- **感知机**：通常使用阶跃函数（sign函数）
- **多层感知机**：使用连续可微的激活函数（如sigmoid、tanh、ReLU等），便于梯度下降优化

### 为什么会有这样的区别？

**根本原因**：线性模型的局限性

感知机的决策边界是一条直线（或超平面），其数学表达为：
```
y = sign(w·x + b)
```

而多层感知机通过叠加多个非线性变换，可以拟合任意复杂的函数：
```
h = f(W₁·x + b₁)  # 隐层
y = f(W₂·h + b₂)  # 输出层
```

**理论依据**：万能逼近定理（Universal Approximation Theorem）证明，只要有足够的隐层神经元，多层感知机可以以任意精度逼近任何连续函数。

## 2. 假设有一个只有一个隐层的多层感知机，其输入、隐层、输出层的神经元个数分别为 33、512、10，那么这个多层感知机中总共有多少个参数是可以被训练的？

### 参数计算：

**第一层（输入层→隐层）：**
- 权重矩阵：33 × 512 = 16,896 个权重
- 偏置向量：512 个偏置
- 小计：16,896 + 512 = 17,408 个参数

**第二层（隐层→输出层）：**
- 权重矩阵：512 × 10 = 5,120 个权重
- 偏置向量：10 个偏置
- 小计：5,120 + 10 = 5,130 个参数

**总参数数量：**
```
总参数 = (33 × 512 + 512) + (512 × 10 + 10)
       = 16,896 + 512 + 5,120 + 10
       = 22,538 个参数
```

### 答案：该多层感知机共有 **22,538** 个可训练参数。

## 3. 反向传播中，神经元的梯度是如何计算的？权重是如何更新的？

### 梯度计算（链式法则）：

反向传播基于**链式法则**从输出层向输入层逐层计算梯度。

**（1）输出层梯度：**
对于损失函数 L 和输出层激活值 oᵢ：
```
∂L/∂zᵢ = ∂L/∂oᵢ · ∂oᵢ/∂zᵢ
```
其中 zᵢ 是输出层第 i 个神经元的加权输入。

**（2）隐层梯度：**
对于第 l 层的神经元 j：
```
∂L/∂zⱼˡ = (∂L/∂zˡ⁺¹ · Wˡ⁺¹) · ∂aⱼˡ/∂zⱼˡ
         = Σₖ(∂L/∂zₖˡ⁺¹ · wⱼₖˡ⁺¹) · f'(zⱼˡ)
```
这表示当前层的梯度等于下一层梯度的加权和乘以激活函数的导数。

**（3）权重梯度：**
```
∂L/∂wᵢⱼˡ = ∂L/∂zⱼˡ · aᵢˡ⁻¹
```
即权重梯度等于该神经元的梯度乘以上一层神经元的激活值。

### 权重更新：

使用梯度下降法更新权重：

```
wᵢⱼˡ(t+1) = wᵢⱼˡ(t) - η · ∂L/∂wᵢⱼˡ
bⱼˡ(t+1) = bⱼˡ(t) - η · ∂L/∂bⱼˡ
```

其中：
- η 是学习率（learning rate）
- t 表示迭代次数

**常用优化改进：**
- **动量法**：引入速度项 v = μv - η∇L
- **Adam**：自适应学习率，结合动量和一阶矩估计

## 4. 请简述三种避免过拟合问题的方法。

### 方法一：正则化（Regularization）

**L1 正则化**：
```python
L = L₀ + λΣ|wᵢ|
```
倾向于产生稀疏权重，自动进行特征选择。

**L2 正则化**（权重衰减）：
```python
L = L₀ + λΣwᵢ²
```
惩罚大权重，使权重分布更平滑，防止模型对特定特征过度敏感。

### 方法二：Dropout

在训练过程中，以概率 p 随机"丢弃"（置零）部分神经元，使其不参与前向传播和反向传播。

**工作原理**：
- 训练时：每个神经元以概率 p 被随机丢弃
- 测试时：使用所有神经元，但权重乘以 (1-p)

**效果**：
- 强制网络学习冗余表示
- 减少神经元之间的共适应
- 相当于训练多个子网络的集成模型

### 方法三：早停法（Early Stopping）

在验证集上监控模型性能，当验证集损失不再下降时停止训练。

**实施步骤**：
1. 将数据集分为训练集、验证集和测试集
2. 每个epoch后在验证集上评估性能
3. 记录验证集损失最低的模型参数
4. 当验证集损失连续 N 个epoch没有改善时停止训练
5. 恢复验证集性能最好的模型参数

**优点**：
- 防止模型过度拟合训练数据
- 自动选择合适的训练轮次
- 不需要修改网络结构

## 5. sigmoid激活函数的极限是0和1，请给出它的导数形式并求出其在原点的导数值。

### Sigmoid函数定义：

```
σ(x) = 1 / (1 + e⁻ˣ)
```

**性质：**
- 当 x → +∞ 时，σ(x) → 1
- 当 x → -∞ 时，σ(x) → 0
- 在 x = 0 处，σ(0) = 0.5

### 导数推导：

使用商的导数法则：

```
σ'(x) = d/dx [1 / (1 + e⁻ˣ)]
      = -(1 + e⁻ˣ)⁻² · d/dx(1 + e⁻ˣ)
      = -(1 + e⁻ˣ)⁻² · (-e⁻ˣ)
      = e⁻ˣ / (1 + e⁻ˣ)²
```

**简化形式**：

```
σ'(x) = σ(x) · (1 - σ(x))
```

证明：
```
σ'(x) = e⁻ˣ / (1 + e⁻ˣ)²
      = [1 / (1 + e⁻ˣ)] · [e⁻ˣ / (1 + e⁻ˣ)]
      = σ(x) · [(1 + e⁻ˣ - 1) / (1 + e⁻ˣ)]
      = σ(x) · [1 - 1/(1 + e⁻ˣ)]
      = σ(x) · (1 - σ(x))
```

### 在原点的导数值：

```
σ'(0) = σ(0) · (1 - σ(0))
      = 0.5 · (1 - 0.5)
      = 0.5 · 0.5
      = 0.25
```

### 答案：
- **导数形式**：σ'(x) = σ(x) · (1 - σ(x)) = e⁻ˣ / (1 + e⁻ˣ)²
- **原点导数值**：σ'(0) = **0.25**

## 6. 假设激活函数的表达式为 φ(v) = v / √(1 + v²)，请给出它的导数表达式并求出其在原点的取值。

### 激活函数定义：

```
φ(v) = v / √(1 + v²) = v · (1 + v²)^(-1/2)
```

### 导数推导：

**方法一：使用商的导数法则**

设 u = v，w = (1 + v²)^(1/2)，则 φ = u/w

```
φ'(v) = (u'w - uw') / w²
```

计算各项：
- u' = 1
- w' = (1/2)(1 + v²)^(-1/2) · 2v = v(1 + v²)^(-1/2)

代入：
```
φ'(v) = [(1) · (1 + v²)^(1/2) - v · v(1 + v²)^(-1/2)] / (1 + v²)
      = [(1 + v²)^(1/2) - v²(1 + v²)^(-1/2)] / (1 + v²)
      = [(1 + v²) - v²] / [(1 + v²)^(1/2) · (1 + v²)]
      = 1 / (1 + v²)^(3/2)
```

**方法二：使用乘法法则和链式法则**

```
φ'(v) = d/dv [v · (1 + v²)^(-1/2)]
      = (1) · (1 + v²)^(-1/2) + v · d/dv[(1 + v²)^(-1/2)]
      = (1 + v²)^(-1/2) + v · (-1/2)(1 + v²)^(-3/2) · 2v
      = (1 + v²)^(-1/2) - v²(1 + v²)^(-3/2)
      = [(1 + v²) - v²] / (1 + v²)^(3/2)
      = 1 / (1 + v²)^(3/2)
```

### 在原点的导数值：

```
φ'(0) = 1 / (1 + 0²)^(3/2)
      = 1 / 1^(3/2)
      = 1 / 1
      = 1
```

### 答案：
- **导数表达式**：φ'(v) = **1 / (1 + v²)^(3/2)**
- **原点导数值**：φ'(0) = **1**

### 补充说明：

这个激活函数的性质：
- **奇函数**：φ(-v) = -φ(v)，关于原点对称
- **有界性**：|φ(v)| < 1，当 v → ±∞ 时，φ(v) → ±1
- **光滑性**：处处可微，导数连续
- **原点线性**：在原点附近近似为 φ(v) ≈ v（类似恒等映射）
- **非饱和性**：与 sigmoid 不同，这个函数在整个定义域上梯度都不为零，有利于梯度传播

这是一个实用的激活函数，类似于 **SiLU (Sigmoid Linear Unit)** 或 **Swish** 的变体，在某些深度学习任务中表现良好。

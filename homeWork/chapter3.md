## 1. 计算AlexNet、VGG19、ResNet152三个网络中的神经元数目及可训练的参数数目

### AlexNet (2012)
- **神经元数目**: 约 650,000 个神经元
- **可训练参数**: 约 60,000,000 (60M) 个参数
- **架构**: 5个卷积层 + 3个全连接层
  - 全连接层: FC1(4096神经元) + FC2(4096神经元) + FC3(1000神经元)

### VGG19 (2014)
- **神经元数目**: 由于全连接层较大,约包含数百万神经元
- **可训练参数**: 约 143,000,000 (143M) 个参数
- **架构**: 16个卷积层 + 3个全连接层
  - 所有卷积核使用3×3
  - 全连接层: FC1(4096) + FC2(4096) + FC3(1000)

### ResNet152 (2015)
- **神经元数目**: 由于使用瓶颈设计,神经元数量相对较少
- **可训练参数**: 约 60,419,944 (60.4M) 个参数
- **架构**: 152层深度网络
  - 使用残差连接和瓶颈设计
  - 最后全连接层: 1000个神经元(ImageNet分类)

## 2. 简述错误率与IoU、mAP的关系

### IoU (Intersection over Union)
- IoU用于衡量预测边界框与真实边界框的重叠程度
- 通常设置阈值(如0.5),当IoU≥阈值时认为预测正确(True Positive)
- **关系**: IoU越高,定位精度越高,错误率越低

### mAP (mean Average Precision)
- mAP是所有类别的平均精度(AP)的平均值
- 计算AP时需要基于IoU阈值来判断预测是否正确
- **关系**:
  - mAP综合考虑了precision(精确率)和recall(召回率)
  - 错误率 = 1 - 准确率
  - mAP越高,模型整体性能越好,错误率越低

### 三者关系总结
```
错误率 ↑ ⇔ 准确率 ↓ ⇔ mAP ↓
IoU ↑ ⇒ 定位更准确 ⇒ True Positive增加 ⇒ mAP ↑ ⇒ 错误率 ↓
```

## 3. 简述训练过程中收敛、训练精度和测试精度之间的关系

### 基本概念
- **收敛**: 损失函数趋于稳定,参数更新幅度减小
- **训练精度**: 模型在训练集上的准确率
- **测试精度**: 模型在测试集上的准确率(泛化能力)

### 三种典型情况

#### (1) 欠拟合(Underfitting)
- **特征**: 训练精度低,测试精度低
- **原因**: 模型过于简单,无法学习数据特征
- **解决**: 增加模型复杂度、增加特征

#### (2) 良好拟合(Good Fit)
- **特征**: 训练精度高,测试精度高,两者接近
- **状态**: 模型收敛且泛化能力强
- **目标**: 这是理想的训练状态

#### (3) 过拟合(Overfitting)
- **特征**: 训练精度高,测试精度低,两者差距大
- **原因**: 模型过于复杂,记住了训练数据的噪声
- **解决**: 增加数据、正则化、Dropout、早停等

### 收敛与精度的关系
```
训练初期: 欠拟合 → 训练精度和测试精度都上升
训练中期: 逐渐收敛 → 训练精度和测试差距缩小
训练后期:
  - 良好收敛: 两者都高且接近
  - 过拟合: 训练精度继续上升,测试精度下降
```

## 4. 简述图像风格迁移应用的基本过程

### 核心思想
将一张图像的内容(Content)与另一张图像的风格(Style)融合,生成新的图像。

### 基本过程

#### 步骤1: 选择预训练网络
- 使用预训练的CNN(如VGG19)作为特征提取器
- 网络的不同层捕获不同级别的特征
  - 浅层: 低级特征(纹理、边缘)
  - 深层: 高级特征(内容、结构)

#### 步骤2: 提取特征
- **内容图像(C)**: 提取深层特征作为内容表示
- **风格图像(S)**: 使用Gram矩阵提取多层特征的风格表示
- **生成图像(G)**: 初始化为内容图像或随机噪声

#### 步骤3: 定义损失函数
总损失 = α × 内容损失 + β × 风格损失

- **内容损失**: 生成图与内容图在指定层特征的均方误差
  ```
  L_content = ||F(G) - F(C)||²
  ```

- **风格损失**: 生成图与风格图Gram矩阵的差异
  ```
  L_style = ||Gram(G) - Gram(S)||²
  ```

#### 步骤4: 优化生成图像
- 使用梯度下降(L-BFGS或Adam)优化生成图像的像素值
- 保持网络参数不变,只更新生成图像
- 迭代 minimizing 总损失

#### 步骤5: 输出结果
- 经过数百/数千次迭代后
- 生成图像保留了内容图的结构
- 同时呈现了风格图的纹理、色彩和艺术风格

### 快速风格迁移(实时版本)
- 训练一个"变换网络"直接生成风格化图像
- 前向传播一次即可完成风格迁移
- 速度: 从数十分钟降至数十毫秒

### 应用场景
- 艺术创作(照片转名画风格)
- 图像编辑与美化
- 视频风格迁移
- 商业应用(如Prisma app)

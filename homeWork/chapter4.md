# VGG19 网络池化层输出形状计算

## 输入规格
- Batch size: 1
- 输入尺寸: (224, 224, 3)
- 输入形状: `(1, 224, 224, 3)` - `(batch, height, width, channels)`

## VGG19 网络结构与池化后形状

### Block 1
- **Conv1-1**: 64 filters, 3×3, padding='same' → `(1, 224, 224, 64)`
- **Conv1-2**: 64 filters, 3×3, padding='same' → `(1, 224, 224, 64)`
- **MaxPool1**: 2×2 pool, stride=2 → **`(1, 112, 112, 64)`** ✅

### Block 2
- **Conv2-1**: 128 filters, 3×3, padding='same' → `(1, 112, 112, 128)`
- **Conv2-2**: 128 filters, 3×3, padding='same' → `(1, 112, 112, 128)`
- **MaxPool2**: 2×2 pool, stride=2 → **`(1, 56, 56, 128)`** ✅

### Block 3
- **Conv3-1**: 256 filters, 3×3, padding='same' → `(1, 56, 56, 256)`
- **Conv3-2**: 256 filters, 3×3, padding='same' → `(1, 56, 56, 256)`
- **Conv3-3**: 256 filters, 3×3, padding='same' → `(1, 56, 56, 256)`
- **Conv3-4**: 256 filters, 3×3, padding='same' → `(1, 56, 56, 256)`
- **MaxPool3**: 2×2 pool, stride=2 → **`(1, 28, 28, 256)`** ✅

### Block 4
- **Conv4-1**: 512 filters, 3×3, padding='same' → `(1, 28, 28, 512)`
- **Conv4-2**: 512 filters, 3×3, padding='same' → `(1, 28, 28, 512)`
- **Conv4-3**: 512 filters, 3×3, padding='same' → `(1, 28, 28, 512)`
- **Conv4-4**: 512 filters, 3×3, padding='same' → `(1, 28, 28, 512)`
- **MaxPool4**: 2×2 pool, stride=2 → **`(1, 14, 14, 512)`** ✅

### Block 5
- **Conv5-1**: 512 filters, 3×3, padding='same' → `(1, 14, 14, 512)`
- **Conv5-2**: 512 filters, 3×3, padding='same' → `(1, 14, 14, 512)`
- **Conv5-3**: 512 filters, 3×3, padding='same' → `(1, 14, 14, 512)`
- **Conv5-4**: 512 filters, 3×3, padding='same' → `(1, 14, 14, 512)`
- **MaxPool5**: 2×2 pool, stride=2 → **`(1, 7, 7, 512)`** ✅

## 池化层输出形状总结

| 池化层 | 输出形状 (NCHW格式) | 输出形状 (NHWC格式) | 特征图尺寸 | 通道数 |
|--------|---------------------|---------------------|------------|--------|
| Pool1  | `(1, 64, 112, 112)` | `(1, 112, 112, 64)` | 112×112 | 64 |
| Pool2  | `(1, 128, 56, 56)`  | `(1, 56, 56, 128)`  | 56×56   | 128 |
| Pool3  | `(1, 256, 28, 28)`  | `(1, 28, 28, 256)`  | 28×28   | 256 |
| Pool4  | `(1, 512, 14, 14)`  | `(1, 14, 14, 512)`  | 14×14   | 512 |
| Pool5  | `(1, 512, 7, 7)`    | `(1, 7, 7, 512)`    | 7×7     | 512 |

## 计算说明

1. **卷积层**: 所有卷积层使用 3×3 卷积核，padding='same'，保持空间尺寸不变
2. **池化层**: 所有池化层使用 2×2 最大池化，stride=2，空间尺寸减半
3. **通道数变化**: 64 → 128 → 256 → 512 → 512
4. **空间尺寸变化**: 224 → 112 → 56 → 28 → 14 → 7

## 参数量说明

VGG19 总参数量约为 143,667,240 个参数，其中大部分参数集中在后几个卷积块中。
